# 搬屎贰号
- 集百家之所短，可搬图文 视频 聊天记录
- 可缓存可查重
- 去掉沟槽的评分限制和时间限制
-壹号地址 https://github.com/anka-afk/astrbot_sowing_discord

## ✨ 功能

- 🚫 自动从指定的群聊(推荐加入 1059470620 或者 1057666434, 共建搬史中心), 转发聊天记录(史)到其他指定的群聊(详见设置)
1. 加入天天发史的群聊
2. 设置该群聊为搬史源头, 设置目标群聊
3. bot 开始全自动搬史!
